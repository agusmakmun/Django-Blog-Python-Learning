About

Python Learning - Sengaja dibuat untuk memperluas jaringan pengetahuan tentang bahasa permograman Python. dengan demikian akan mempermudah para programmer untuk mendapatkan knowlage. Terdapat juga mengenai masalah kuliah yang berhubungan dengan Python Programming.

Sejarah

Python Learning - Telah publish sejak awal tahun 2014 oleh salah satu mahasiswa UMS di Surakarta (Solo) Indonesia yang bernama Agus Makmun atau yang akrab dipanggil "Summon Agus". Pada mulanya, website Python Learning adalah website yang berbasis “asking a question” (tanya jawab). namun karena suatu hal dan lumayan lama tidak terurus, sehingga Python Learning kembali publish dengan nuansa yang berbeda.

Redesign

Python Learning - Telah Redesign sebanyak 2 kali, redesign pertama menggunakan cms Wordpress. Alhamdulillah kali ini telah sesuai dengan cita-cita pendiri yang dari dahulu ingin menggunakan Django sebagai framework-nya. Agar sesuai dengan tema yang diangkat, yaitu mengenai Belajar Python Programming.
